◆ UDKHalo()

UDKHalo::UDKHalo ( )

Frame setup.

Sets and populates menubar with File and Help menus. File contains Hello and
Quit entries while Help contains About entry. A status bar at the bottom is
also created. A relevant bind to the OnHello, OnExit, and OnAbout routines
is done for desired callback.

wxAuiManager* UDKHalo::m_PaneManager

Prepare the manager for various supported features panes.

Do the necessary adding and modification of feature panes

See also
UDKHalo::UDKHalo()

For managing varitey of panes or panels

Chiefly used for adding, handling resizing, docking, and more
of utility panes (to be) present in UDKHalo.
wxAuiManager works as follows: the programmer adds panes to the
class, or makes changes to existing pane properties (dock
position, floating state, show state, etc.). To apply these
changes, wxAuiManager's Update() function is called. This batch
processing can be used to avoid flicker, by modifying more than
one pane at a time, and then "committing" all of the changes at
once by calling Update().
