)

Seeks to the specified position.

UDK's implementation of Seek function. Although wxFile::Seek is present
which is "move ptr ofs bytes related to start/current offset/end of file"
This routine, however, correctly sets the m_Getptr and m_Putptr to ofs for
later use.

Parameters
ofsOffset to seek to
modeParameter indicating how file offset should be interpreted. wxFromStart: Seek from the file beginning wxFromEnd: Seek from the current position wxFromCurrent Seek from end of the file
Returns
The actual offset position achieved
